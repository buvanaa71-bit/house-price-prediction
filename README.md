🏠 House Price Prediction — Predictive Analytics

A complete Machine Learning project that predicts 
house prices using Linear Regression.

Built as part of an AI & Data Science Internship.

📊 Dataset    : 1,000 realistic house records
🤖 Algorithm  : Linear Regression
📈 Accuracy   : R² = 0.87 (87%)
🐍 Language   : Python (Google Colab)

🔧 Libraries Used:
- Pandas & NumPy (data processing)
- Matplotlib & Seaborn (visualizations)
- Scikit-learn (ML model)

- Complete Python notebook
